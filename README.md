Azure AI Powered Academic Risk Intelligence Platform

## Problem
Students often fail without early academic risk detection.

## Solution
EduRisk AI predicts student academic risk and generates
personalized 7-day improvement plans using Microsoft AI.

## Microsoft Technologies Used
- Azure Machine Learning (Risk Prediction)
- Azure OpenAI Service (Personalized Guidance)
- Azure Key Vault (Secure Secrets Management)
- Streamlit (Web Interface)

## How It Works
1. User enters academic metrics
2. Azure ML predicts risk level
3. Azure OpenAI generates action plan
4. Dashboard visualizes insights
